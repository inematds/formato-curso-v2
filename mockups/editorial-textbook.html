1.Veja a anatomia de um prompt

Um prompt não é uma pergunta solta: é um pequeno briefing1. Quando o resultado vem errado, quase sempre falta uma das cinco peças que o modelo precisa para decidir o que fazer.

Antes de reescrever a instrução inteira, vale uma pausa: leia a saída de volta e pergunte qual peça você não entregou. Quase nunca é a redação — é uma informação ausente. Nas próximas seções desmontamos cada peça e remontamos um prompt do zero.
